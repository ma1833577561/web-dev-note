## 1、基础使用

```bash

    # 新建 文件
    %: touch [文件+后缀]

    # 新建目录
    %: mkdir [文件]

    # 删除 
    %: rm -rf [文件、文件夹]

    # 查看
    %: ls

    # 编辑文件内容 (英文模式输入命令)
    %: vim [文件+后缀]
    # 按下【i键】进入输入模式，移动光标输入内容即可
    # 退出，按【esc键】退出编辑模式, 同时按下【shift/⬆️】+【:】键，输入【wq】+【回车件】保存并退出

    # 权限不够
    %: sudo [命令]
    # 再输入锁屏密码就行啦

```
